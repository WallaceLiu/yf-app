
# **YF 小说app Android 客户端项目简析** #

*注：本文假设你已经有Android开发环境*

本文以eclipse为例<br>
启动Eclipse，导入Android客户端项目，请确保你当前的Android SDK是最新版。
如果编译出错，请修改项目根目录下的 project.properties 文件。
推荐使用Android 4.0 以上版本的SDK：<br>

target=android-14
